# License Console UI Demo

基于参考页面重构的安全演示项目：Next.js App Router、shadcn/ui 组件结构、Nano Stores、Biome、Tailwind CSS。

> 说明：项目不会生成或签名任何真实 JetBrains 许可证。`/api/generate-demo` 只返回带 `DEMO-` 前缀的无效演示字符串，用于验证表单、状态管理、复制等交互。

## 运行

```bash
pnpm install
pnpm dev
```

访问 `http://localhost:8088`。

## 命令

```bash
pnpm check
pnpm check:fix
pnpm build
pnpm start
```
